Sistema de Evaluación Docente de la Universidad Montrer

La universidad Montrer hacía un año manejaba un sistema de evaluación docente presencial, en el cual los Directores de área de cada materia llenaban a mano en un documento de Excel los datos de materias, profesores y grupos, esto se hacía muy tardado, por lo que se solicitó hacer un sistema que automatizara todo esto.

A nosotros como estudiantes de Ingeniaría de Software en la materia de desarrollo ágil se nos encargó la realizción del proyecto, al finalizar el tiempo de entrega se subió el sistema desarrollado a un servidor que las personas de la universidad Montrer rentan para administrar su sitio web, al subirse hubo pequeños inconvenientes de compativilidad que se solucionaron sin retrasar la entrega, al momento de utilizar el sistema por primera vez se obtuvo una aprovación de que el sistema cumplia los requerimientos solicitados.

A continuación se puede ver la página del sistema para una mejor visualización, sin embargo como es un sistema cuyas credenciales de acceso son privadas no se puede tener acceso a todo el sistema, únicamente se puede apreciar el LogIn para ver el diseño del sitio.
